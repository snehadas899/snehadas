# Hospital Emergency Room Dashboard

## Project Overview
This project analyzes hospital emergency room data using Microsoft Excel to track patient flow and key performance indicators.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboard Design

## Key KPIs
- Total Patients
- Average Waiting Time
- Patient Satisfaction Score
- Admission Status
- Department Referrals
- Gender and Age Analysis

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- KPI Reporting
- Dashboard Creation
